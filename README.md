# Ten Temples Server


#### Spielregeln

Pro Runde ein Angriff und ein Konter

Runde 1, S1 fordert heraus
-	S1 Angriff
	-	Offense P1_A1-P1_A3
	-	Defense P1_D1-P1_D3 <-- P2_A1-P2_A3
-	S2 Gegenangriff
	-	Defense P2_D1-P2_D3 --> P1_A1-P1_A3
	-	Offense P2_A1-P2_A3

= Sieger Angriff
= Sieger Gegenangriff 
= Sieger Runde
